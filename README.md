## 🎉 사소해(Sasohae) 소개
### 🎁 선물은 어떤걸 주는게 좋을까?
> - 부모님,애인,친구에게 어떤 선물을 주는게 좋을까 고민될때 해결해보세요.

### 🍔 오늘 점심,저녁은 뭐먹지?
> - 오늘 점심 또는 저녁에 먹을메뉴가 고민이될때 해결해보세요.
 
### 💲 축의금은 얼마내야할까...?
> - 축의금을 얼마낼지 고민이될때 해결해보세요.

### 😣 요즘 고민도많고 힘드네...
> - 어디가서 말못할 고민들을 남겨보고 함께 공감해보세요.
